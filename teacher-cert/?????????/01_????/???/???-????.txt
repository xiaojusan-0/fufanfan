教师观：科目一《综合素质》备考笔记

整理日期：2026-07-03
资料来源：今日教师观思维导图截图


一、考点定位

所属模块：
科目一《综合素质》- 职业理念

常见题型：
材料分析题、单选题

核心：
新课程改革背景下的教师观，主要考“教师角色转变”和“教师教学行为转变”。


二、教师职业角色的转变

口诀：
见醋放盐

可以理解为：

1. 促：学生学习的引导者和学生发展的促进者
2. 建：课程的建设者和开发者
3. 研：教育教学的研究者
4. 放：社区型的开放教师


1. 学生学习的引导者和学生发展的促进者

角度：
从教师与学生关系看。

含义：
教师不只是讲知识的人，更要引导学生学习，促进学生发展。

关键词：

- 引导
- 促进
- 启发
- 帮助学生进步
- 关注学生发展

材料识别：
如果材料中老师帮助学生找到方法、激发兴趣、促进进步，可以写这一点。


2. 课程的建设者和开发者

角度：
从教学与课程的关系看。

含义：
教师不能只照本宣科，还要开发和利用课程资源，让教学内容更丰富。

关键词：

- 开发课程资源
- 丰富教学内容
- 联系生活
- 整合资源
- 创造性使用教材

材料识别：
如果材料中老师补充生活案例、开发校本课程、整合资料，可以写这一点。


3. 教育教学的研究者

角度：
从教学与研究的关系看。

含义：
教师要研究教育教学问题，在反思和研究中改进教学。

关键词：

- 研究教学问题
- 反思教学
- 行动研究
- 改进教学
- 发现问题并解决问题

材料识别：
如果材料中老师课后反思、记录问题、研究学生学习情况，可以写这一点。


4. 社区型的开放教师

角度：
从学校与社区的关系看。

含义：
教师的工作不只在课堂和学校内，也要联系家庭、社区和社会资源。

关键词：

- 社区资源
- 家校合作
- 社会实践
- 服务范围扩大
- 开放教师

材料识别：
如果材料中老师联系家长、社区、博物馆、社会机构等资源，可以写这一点。


三、教师教学行为的转变

口诀：
赞助盒饭

可以理解为：

1. 赞：对待师生关系，强调尊重、赞赏
2. 助：对待教学，强调帮助、引导
3. 合：对待其他教育者，强调合作
4. 反：对待自我，强调反思


1. 对待师生关系：尊重、赞赏

关键词：

- 尊重学生
- 赞赏学生
- 平等对话
- 保护自尊
- 鼓励学生

材料识别：
如果材料中老师肯定学生、尊重学生意见、保护学生自信，可以写这一点。


2. 对待教学：帮助、引导

关键词：

- 帮助
- 引导
- 启发
- 支持学生学习
- 不包办代替

材料识别：
如果材料中老师不是直接给答案，而是引导学生思考，可以写这一点。


3. 对待自我：反思

关键词：

- 教学反思
- 总结经验
- 改进教学
- 发现不足

材料识别：
如果材料中老师课后反思、修改教学方法、总结不足，可以写这一点。


4. 对待其他教育者：合作

关键词：

- 同事合作
- 家校合作
- 教研合作
- 与其他教师共同备课

材料识别：
如果材料中老师与同事、家长或其他教育者配合，可以写这一点。


四、教师专业素养

教师专业素养包括：

1. 专业知识
2. 专业精神
3. 专业能力
4. 专业实践

教师专业发展的常见方式：

1. 专家引领
2. 同伴互助
3. 校本研修
4. 微格教学
5. 行动研究
6. 教学反思


五、习近平关于教师的重要表述

1. 四有好老师

- 有理想信念
- 有道德情操
- 有扎实学识
- 有仁爱之心

2. 四个引路人

- 做学生锤炼品格的引路人
- 做学生学习知识的引路人
- 做学生创新思维的引路人
- 做学生奉献祖国的引路人

3. 四个相统一

- 教书和育人相统一
- 言传和身教相统一
- 潜心问道和关注社会相统一
- 学术自由和学术规范相统一


六、材料分析题答题模板

题目问：
请从教师观角度评价材料中教师的行为。

可以这样写：

材料中教师的做法符合新课程改革背景下的教师观，值得肯定。

第一，教师是学生学习的引导者和学生发展的促进者。教师不仅要传授知识，还要引导学生学习，促进学生发展。材料中，教师......，体现了这一点。

第二，教师是课程的建设者和开发者。教师应创造性地使用教材，开发课程资源，丰富教学内容。材料中，教师......，体现了这一点。

第三，教师是教育教学的研究者。教师应在教学中发现问题、研究问题，并通过反思不断改进教学。材料中，教师......，体现了这一点。

第四，教师是社区型的开放教师。教师应加强与家庭、社区和社会资源的联系，拓展教育空间。材料中，教师......，体现了这一点。

第五，教师的教学行为也应发生转变。教师应尊重赞赏学生，帮助引导学生，反思自身教学，并与其他教育者合作。材料中，教师......，体现了这一点。

综上，该教师的做法符合教师观，有利于学生发展和教师专业成长。


七、反面材料怎么写

如果材料中教师做法不好，可以这样写：

材料中教师的行为违背了新课程改革背景下的教师观。

常见反面点：

1. 只灌输知识，没有引导学生主动学习。
2. 只照本宣科，没有开发课程资源。
3. 不反思教学，不研究问题。
4. 不联系家庭、社区和社会资源。
5. 不尊重学生，只批评打压。
6. 不与同事、家长合作。


八、最容易混淆的地方

1. “老师帮助学生进步”
对应：促进者、引导者。

2. “老师补充生活资源、开发课程”
对应：课程建设者和开发者。

3. “老师研究教学问题”
对应：教育教学研究者。

4. “老师联系家长、社区”
对应：社区型开放教师。

5. “老师尊重、表扬学生”
对应：教学行为转变中的尊重、赞赏。

6. “老师课后总结改进”
对应：教学行为转变中的反思。


九、材料分析题 8 条填空套话

使用方法：
看到材料后，先判断它落在哪一条，再把老师的具体做法填进横线处。正面材料写“成为了”，反面材料写“没有成为”或“没有做到”。


1. 学生学习的引导者和学生发展的促进者

从教师与学生的关系看，新课程要求教师应该是学生学习的引导者和发展的促进者。教师不仅要积极引导学生的学习和发展，还要注重培养学生健康的人格和良好的品行。

材料中，老师__________，成为了/没有成为学生的引导者和促进者。


2. 课程的建设者和开发者

从教学与课程的关系看，新课程要求教师应该是课程的建设者和开发者。教师要积极参与课程的建设工作，灵活使用教材，并能够通过自制教学工具、引入课外资源等方式丰富课堂。

材料中，老师__________，成为了/没有成为课程的建设者和开发者。


3. 教育教学的研究者

从教学与研究的关系看，新课程要求教师应该是教育教学的研究者。教师要以研究者的眼光审视教学中出现的各种问题，并对出现的问题进行探索和研究。

材料中，老师__________，成为了/没有成为教学的研究者。


4. 社区型的开放教师

从学校与社区的关系看，新课程要求教师应该是社区型的开放教师。教师不仅是学校的一员，而且是整个社区的一员，是整个社区教育、文化事业建设的共建者。

材料中，老师__________，成为了/没有成为社区型的开放教师。


5. 尊重、赞赏

在对待师生关系上，新课程强调尊重、赞赏。教师不仅要保护学生的自尊心，还要学会对学生加以赞赏和表扬。

材料中，老师__________，做到了/未做到尊重和赞赏。


6. 帮助、引导

在对待教学上，新课程强调帮助、引导。教师不仅要帮助学生解决问题、克服困难，还要引导学生形成正确的价值观。

材料中，老师__________，做到了/未做到帮助和引导。


7. 反思

在对待自我上，新课程强调反思。教师要不断地通过教育案例、教育故事、教育心得等方式来反观自己的言行，以此逐步提高教学质量。

材料中，老师__________，做到了/未做到自我反思。


8. 合作

在对待与其他教育者的关系上，新课程强调合作。不同学科、不同年级的教师要相互配合，齐心协力地培养学生；除此之外，教师还要与家长处理好关系，加强家校合作。

材料中，老师__________，做到了/未做到合作。
